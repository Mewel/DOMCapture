DOM Capture Help

DOM Capture lets you take screenshots of parts of a webpage. The unique and awesome thing is that you don't have to see the area of the web page you want to capture. It is possible to take a screenshot of the whole website even if only a small part is visible to you.

Be aware that DOM Capture is more of an expert tool. If your unfamiliar with HTML, CSS or the Developer Console you should check some videos first before using this tool.

How to use

Favorites

Lets you store your favorite settings.

Selector

The selector defines which part of the webpage should be captured. It uses the CSS selector syntax. It's the same syntax jQuery uses for selecting elements.

Tip: To easily copy a selector just open the Developer Tools. Go to the element you are interested in. Do a right-click -> Copy -> Copy Selector. This copies the selector for this specific element. Paste the result in the DOM Selector and try it out!

The easiest selector would be the body selector which copies the whole webpage.

If you want to copy multiple elements at once just separate them by comma. For example #content, #footer, ....

Background

Defines the background color of the resulting canvas. The default value is the background color of the body.

Multi Element Layout

Defines how multiple elements are merged together. There are three options:

  • Keep (default): Keeps the layout of the whole webpage. The resulting canvas will have the same size as the whole body. Elements are put on their original position.
  • Vertical: Merges the elements vertically.
  • Horizontal: Merges the elements horizontally.

DOM Traversal

Defines the order in which multiple elements are traversed. There are two options:

  • Document (default): Traverse the DOM in document order.
  • User: Traverse the DOM in the order given by the user. The comma separated list is executed in this order.
The DOM Traversal setting is only important if multiple elements are selected (separated by comma) and if the Multi Element Layout setting is set to vertical or horizontal.

Example: Open a wikipedia page for example this one. As selector use #mw-panel, #content and set the Multi Element Layout to horizontal. Now you can switch between Document and User DOM Traversal to see the difference.

On each wikipedia page the #mw-panel div is placed before the #content div, which is kind of unusual. To get the right snapshot you have to use the User DOM Traversal
